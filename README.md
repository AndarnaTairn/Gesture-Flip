# Gesture-Flip

A browser-based PDF viewer that allows users to upload documents and navigate them using voice commands, enabling comfortable hands-free reading.

## Problem
Using phones while lying down for long periods can cause strain and discomfort. This project offers a laptop-based alternative for reading PDFs without constant manual interaction.

## Features
- Upload and read any PDF in browser
- Voice commands for:
  - Next page
  - Previous page
  - Rotate PDF
- Hands-free navigation
- Designed for comfortable reading while lying down
- Accessibility-focused interaction

## Tech Stack
- HTML
- CSS
- JavaScript
- Web Speech API
- PDF.js

## How It Works
The application loads user-uploaded PDFs and uses voice recognition to interpret commands for document navigation and orientation control in real time.

## Use Cases
- Hands-free studying
- Accessibility support
- Comfortable long-form reading
- Touchless document interaction

## Future Improvements
- Custom voice shortcuts
- Bookmark commands
- Gesture + voice hybrid control
